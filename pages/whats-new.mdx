# What's New

## Color Palette

Color Palette now shows sorted colors, bringing a new level of organization and efficiency to your design exploration.

## Move Element

We've rewritten the Move Element feature to use CSS transform translate instead of position relative to move things. This overhaul makes moving elements smoother and more intuitive than ever.

## Extract Images

We've added image size info to the Extract Images feature. Now, you can see the size of any image immediately, along with image dimensions and image format, simplifying your workflow and decision-making process.

## Page Navigation

We've fixed an important issue to enhance page navigation. Now, you can easily view the tagname of an element even when it's positioned at the edge of the page.
