# Page Ruler

Page Ruler feature allows you to measure distances between elements on any website in real time. You can also select custom areas to measure, and it works seamlessly with all types of elements, including images, videos, and moving elements.

![Page Ruler](/features/page-ruler.png)

## Use Cases

- **Design Fine-Tuning**: Verify that your elements are correctly spaced.
- **Layout Analysis**: Get insights into the distances between various elements for better layout planning.
- **Responsive Testing**: Measure how elements stack or align at different screen sizes.
- **Media Content**: Use it to measure sizes of images, videos, or any other media types.
