# Extract Images

Extract Images feature lets you effortlessly export all types of images from any website in a click including meta images like OG and Twitter images, background images, standard `<img>` images, srcset images, inline SVGs, canvas images, video posters, and favicons. You can either export all images at once or export images individually.

![Extract Images](/features/extract-images.png)

## Use Cases

- **Web Development**: Easily extract images from any website for use in your projects.
- **Design Analysis**: Assess the variety and type of images used on a website.
- **SEO Audit**: Check meta images like OG and Twitter images for compliance with SEO best practices.
- **Asset Backup**: Create a backup of all visual elements on a website for archival purposes.
