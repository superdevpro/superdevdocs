# Extract Images

Extract Images feature allows you to quickly export all types of images from any website in a click including meta images like OG and Twitter images, background images, standard `<img>` images, srcset images, inline SVGs, canvas images, video posters, and favicons. You can export all images at once or export images individually.

## Use Cases

- **Design Inspiration**: Collect images from websites for use on your website.
- **Design Analysis**: Study image usage patterns across different websites.
- **Performance Optimization**: Analyze image sizes and formats for potential improvements.
- **Competitive Research**: Compare image strategies across competitor websites.
- **SEO Analysis**: Examine OG and Twitter meta images for SEO optimization.
- **Responsive Design**: Extract srcset images to understand responsive strategies.
- **Brand Asset Compilation**: Quickly gather all brand images from a website.
- **Print Material Preparation**: Gather high-res images for print publications.
