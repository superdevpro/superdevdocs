# Move Element

Move Element feature allows you to easily reposition different elements on a website to visualize various layouts and designs. If you make a mistake or change your mind, you can effortlessly undo any changes by pressing `Ctrl+Z` or `Cmd+Z` on macOS.

![Move Element](/features/move-element.png)

## Use Cases

- **Design Prototyping**: Rapidly move elements around to test various design layouts without changing the actual code.
- **Demonstrations**: Show clients how repositioned elements will look on their website without affecting the live site.
- **Content Arrangement**: Easily rearrange content blocks, images, or text to visualize different storytelling flows.
- **Conversion Rate Optimization**: Experiment with element placements that could potentially improve conversion rates before making real changes.
