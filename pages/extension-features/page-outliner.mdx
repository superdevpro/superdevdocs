# Page Outliner

Page Outliner feature allows you to outline the entire website or a specific HTML tag effortlessly. You can also choose the outline opacity percentage and outline color like red, green, blue, and random.

![Page Outliner](/features/page-outliner.png)

## Use Cases

- **Debugging Layouts**: Easily spot issues with the layout by outlining elements to see how they stack.
- **Content Strategy**: Quickly identify key HTML elements like headings, paragraphs, or links to evaluate their distribution.
- **Content Review**: Use different colors to annotate or categorize different sections of a website for content review.
- **Demo or Presentation**: Highlight specific parts of a website while presenting or screen recording.
