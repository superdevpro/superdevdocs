# Image Replacer

Image Replacer feature enables you to effortlessly replace any image on a website that uses the `<img>` tag. This allows you to visualize how the website would look with a different image. If you make a mistake, you can easily undo your changes using the keyboard shortcut `Ctrl+Z` or `Cmd+Z` on macOS.

![Image Replacer](/features/image-replacer.png)

## Use Cases

- **Content Testing**: Experiment with different images to see which one better complements your website's design.
- **Bug Fixing**: Quickly replace faulty or broken images during the development process.
- **Creative Exploration**: For web developers and designers who want to test different images without affecting the live website.
