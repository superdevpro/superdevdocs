# Export Element

Export Element feature allows you to effortlessly export your favorite elements from any website directly to CodePen or an HTML file in a single click. Whether it's a complex layout or a simple button, you can quickly export it from any website and apply it on your own website.

![Export Element](/features/export-element.png)

## Use Cases

- **Rapid Prototyping**: Use existing website elements as starting points for new projects.
- **Debugging**: Easily export problematic elements to isolate and troubleshoot issues.
- **Learning**: Export complex elements to study their HTML and CSS in detail.
- **Template Creation**: Export common elements to use as templates for future projects.
