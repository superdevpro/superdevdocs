# Color Palette

Color Palette feature allows you to discover all the colors used on any website effortlessly and view their respective color codes in `Hex`, `RGB`, or `HSL` color format. Furthermore, you have the option to copy these codes to your clipboard or export them for use in future projects.

## Use Cases

- **Design Analysis**: Quickly understand the color scheme of any website.
- **Brand Guideline Creation**: Extract all colors used in a brand's website.
- **Inspiration Gathering**: Collect color palettes from visually appealing websites.
- **Competitor Research**: Analyze color usage across competitor websites.
- **Consistency Check**: Ensure color uniformity across an entire website.
- **Theme Development**: Extract color schemes for creating themes or templates.
- **Content Creation**: Extract colors for digital content, presentations, and more.
