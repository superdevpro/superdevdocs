# Color Palette

Color Palette feature enables you to discover all the colors used on any website effortlessly and view its respective color codes. Furthermore, you have the option to copy these codes to your clipboard or export them for use in future projects.

![Color Palette](/features/color-palette.png)

## Use Cases

- **Design Inspiration**: : Easily identify and use colors schemes of existing websites to apply in your own projects.
- **Consistency in Design**: Ensure a consistent color scheme across multiple projects by easily identifying and copying color codes.
- **Content Creation**: Use Color Palette to extract colors for use in presentations, images, or any other content.
