# Suggested Tools

SuperDev Pro is a handy tool for web developers, but there are additional browser extensions that can help you even more. We've made a list of tools that work well with SuperDev Pro to help you solve more problems and do your work faster.

## Wappalyzer

[Wappalyzer](https://chromewebstore.google.com/detail/gppongmhjkpfnbhagpmjfkannfbllamg) is a technology profiler that shows you what websites are built with. Find out what CMS a website uses, any framework, e-commerce platform, JavaScript libraries, and many more, uncovering over a thousand technologies in dozens of categories.

## Mobile Simulator

[Mobile simulator](https://chromewebstore.google.com/detail/ckejmhbmlajgoklhgbapkiccekfoccmk) is a responsive viewerÂ that allows users to easily simulate a realistic smartphone or tablet in one click, ensuring their websites are responsive across tens of different device sizes.

## Clear Cache

[Clear Cache](https://chromewebstore.google.com/detail/cppjkneekbjaeellbfkmgnhonkkjfpdn) is a simple and powerful way to clear your browser cache with fine-grained control over what gets cleared, which helps you resolve issues related to outdated content and improve browser performance. Its simplicity and effectiveness make it a go-to solution for developers.

## JSON Viewer Pro

[JSON Viewer Pro](https://chromewebstore.google.com/detail/eifflpmocdbdmepbjaopkkhbfmdgijcc) is a completely free extension to visualize JSON responses in awesome tree and chart view with great user experience and options.

## WebDev Checklist

[WebDev Checklist](https://chromewebstore.google.com/detail/iahamcpedabephpcgkeikbclmaljebjp) is the ultimate checklist for all serious web developers building modern websites. It analyzes any web page for violations of best practices, helping web developers easily discover problem areas in websites.
