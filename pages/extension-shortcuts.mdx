# Extension Shortcuts

No matter which operating system you call home - Windows, Linux, or macOS, SuperDev Pro provides shortcuts for things that matter, allowing you to focus on what you do best.

- Open Extension: <code>{findCtrlBtn()}+Shift+S</code>
- Close Extension: <code>{findCtrlBtn()}+Shift+S</code>
- Disable Active Feature: `Escape`
- Navigate Elements: `Up, Down`
- Undo Text Editor Edits: <code>{findCtrlBtn()}+Z</code>
- Undo Moved Element: <code>{findCtrlBtn()}+Z</code>
- Undo Deleted Element: <code>{findCtrlBtn()}+Z</code>
- Undo Replaced Image: <code>{findCtrlBtn()}+Z</code>
